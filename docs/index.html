aimlock

조준점 검색과 제작을 한곳에서 처리하는 허브입니다. 원하는 스타일을 빠르게 찾고, 코드 복사와 공유까지 바로 이어서 진행할 수 있습니다.

조준점 검색

제목, 태그, 작성자 기준으로 검색하고 타입 필터/정렬로 원하는 조준점을 빠르게 찾습니다.

조준점 제작

색상/라인/점 옵션을 조절하며 실시간 미리보기를 확인하고 원하는 조준점을 직접 만듭니다.

프리셋 관리

내 프리셋을 저장하고 불러오며, 랜덤 생성으로 빠르게 다양한 스타일을 시도할 수 있어요.

즐겨찾기 & 코드

로그인 후 즐겨찾기로 모아두고, 검색 결과의 코드 복사/공유로 바로 이어갈 수 있습니다.

자주 쓰는 도구

이용 가이드

기능 흐름을 따라가면 조준점 찾기부터 제작까지 빠르게 연결됩니다.

  1. 1) 조준점 검색
    검색어 입력, 타입 필터, 정렬을 이용해 원하는 조준점을 찾아요.
  2. 2) 즐겨찾기
    로그인 후 자주 쓰는 조준점을 별로 모아두고 빠르게 접근해요.
  3. 3) 조준점 제작
    내 조준점 프리셋을 만들고, 코드로 관리/저장할 수 있어요.
  4. 4) 코드 복사 & 공유
    복사 버튼으로 코드를 가져가고, 공유 버튼으로 링크를 바로 전달해요.

조준점 검색

검색 · 필터 · 정렬 · 즐겨찾기를 사용해 원하는 조준점을 찾아보세요.
